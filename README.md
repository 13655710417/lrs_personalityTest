# 狼人杀人格测试网站

一个基于19道选择题的狼人杀人格测试网站，根据6个维度分析用户在狼人杀游戏中的行为倾向，匹配30种人格结果。

## 功能特性

- 19道精心设计的狼人杀情境选择题
- 6个分析维度：游戏风格、游戏态度、思考方式、社交角色、道德类型、操作风格
- 30种人格结果，配有搞笑真实的描述文案
- 现代化简约界面，响应式设计
- 实时进度显示，测试结果可视化
- 分享功能（模拟）

## 数据说明

### 题目数据 (`data.js`)

包含19道题目，每道题有3-6个选项，每个选项对应6个维度的分数（1-3分或null）。

### 人格结果 (`data.js`)

包含30种人格，每种人格有名称和描述。人格匹配基于用户的前2个最高维度。

### 维度定义

1. **play_style**：激进(1) / 平衡(2) / 保守(3)
2. **game_attitude**：认真(1) / 平衡(2) / 佛系(3)
3. **think_way**：逻辑流(1) / 平衡(2) / 人性流(3)
4. **social_role**：带队(1) / 平衡(2) / 躺平(3)
5. **moral_type**：正派(1) / 平衡(2) / 心机(3)
6. **operate_style**：稳定(1) / 平衡(2) / 整活(3)

### 计分规则

1. 每题选项对应1/2/3分，无关维度不赋值
2. 最终取6个维度平均分（保留1位小数）
3. 取前2个最高分维度作为核心标签
4. 按区间匹配人格：
   - 1.0–1.6 = 偏左（激进/认真/逻辑/带队/正派/稳定）
   - 1.7–2.3 = 平衡
   - 2.4–3.0 = 偏右（保守/佛系/人性/躺平/心机/整活）

## 自定义修改

### 修改题目

编辑 `data.js` 中的 `questions` 数组，遵循现有格式。

### 修改人格结果

编辑 `data.js` 中的 `personalities` 数组。

### 修改样式

编辑 `style.css` 文件，或直接在 `index.html` 中引入其他CSS框架。

### 修改界面文本

编辑 `data.js` 中的 `frontendText` 对象。
